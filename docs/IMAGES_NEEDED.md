# Images to add — Sai Baba temple

Drop each image in **`assets/images/`** using the exact filename below (all
lowercase, `.jpg`). The app already shows it once the file is there — no code
change needed. Until an image is added, that festival shows generated fallback art.

Guidance (same as before): use **licensed / temple-owned / public-domain** images
(this is a public site), aim for **≥ 800 px** on the short side so they look sharp,
and keep files reasonably small (< ~200 KB).
